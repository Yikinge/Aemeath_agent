# 周报整理技能

当用户想回顾或总结一周时，按下面步骤来：

1. 先调用 `search_memory`，分别检索"这周""最近在做的事""完成/进展"等，拉取本周相关记忆与开放回路。
2. 把素材整理成三段：
   - **做了什么**：本周实际推进/完成的事
   - **卡点**：遇到的困难或没解决的
   - **下周计划**：接下来要跟进的（可结合未完成的承诺）
3. 用爱弥斯一贯的口吻收个尾——是朋友帮着捋了捋这一周，不是机器生成报告。别太长，别用生硬的小标题堆砌。

如果记忆里素材太少，就如实说"这周聊得不多，我能想起来的就这些"，并顺口问一句这周还有什么想记下的。
